# game-library-manager

Projeto desenvolvido em Python durante o curso de Análise e Desenvolvimento de Sistemas.

# Sobre o sistema
Aplicação desktop para adicionar e organizar jogos que o usuário pretende jogar.

# Funcionalidades
Cadastro de jogos
Listagem de jogos
Organização da biblioteca pessoal

# Tecnologias utilizadas
Python - JSON
PyCharm
